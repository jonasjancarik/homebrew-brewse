## Installation

```bash
brew tap jonasjancarik/brewse
brew install brewse
```

Or install directly without tapping:

```bash
brew install jonasjancarik/brewse/brewse
```

## Usage

Launch the interactive browser:

```bash
brewse
```

Or search directly:

```bash
brewse python
```
